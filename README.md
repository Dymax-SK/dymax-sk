Welcome to my GitHub profile. I'm a modder of GTA V/LSPDFR and a composer of original classical piano music. This space is a collection of my programming and scripting projects.

## Modding & Development

The project I'm working on right now is **Police Academy**, which is a mod for LSPDFR (GTA V).

**Police Academy** is a mod that adds dynamic elements that teach the player how to do police-related things within legal framework.

* Languages & Tools I know: C#, Java, .NET, LSPDFR and RagePluginHook APIs
* Current Project: Police Academy (in development)

> [!WARNING]  
> The mod has not been published yet.

## About Music

In addition to developing mods, I also compose and publish original classical piano music. Many people find that the concentration required for programming and playing have a lot in common. You can listen to my music on my channel!
